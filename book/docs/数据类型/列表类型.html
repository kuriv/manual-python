列表类型

列表中的元素类型可以不相同，主要用来实现集合类的数据结构。

list = [233, 6.66, True, 4 + 3j, 'Hello World!']
print(list)
print(list[0])
print(list[1:3])
print(list[2:])
print(list[:-1])
print(list * 2)
print(list + [1, 2, 3])

与字符串不同的是，列表中的元素是可以改变的。

list = [233, 6.66, True, 4 + 3j, 'Hello World!']
print(list)
list[2] = False
print(list)
list[1:4] = [1, 2, 3]
print(list)
list[:-1] = []
print(list)

列表也可以进行多个层次的嵌套。

matrix = [[1, 2, 3], [4, 5, 6], [7, 8, 9]]
print(matrix)
print(matrix[0])
print(matrix[1][2])
