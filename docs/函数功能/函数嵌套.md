# 函数嵌套

执行下面的代码，在一个函数中调用另一个函数。

```python
def func1():
    print('Hello World!')


def func2():
    func1()
    print('Hello Python!')


func1()
func2()
```

执行下面的代码，在一个函数中定义另一个函数。

```python
def func1():
    print('Hello World!')

    def func2():
        print('Hello Python!')

    func2()


func1()
```
