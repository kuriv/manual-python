# 类型转换

执行下面的代码，将对象的数据类型显示转换为所需的数据类型。

```python
num1 = int(6.66)
print(num1, type(num1))
num2 = int('233')
print(num2, type(num2))
num3 = float(233)
print(num3, type(num3))
num4 = float('6.66')
print(num4, type(num4))
str1 = str(233)
print(str1, type(str1))
str2 = str(2.33)
print(str2, type(str2))
```

执行下面的代码，运行字符串表达式，并返回表达式的值。

```python
print(eval('233 + 666'))

```
