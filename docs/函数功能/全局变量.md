# 全局变量

函数外部定义的变量在整个文件中有效。

```python
num = 233


def func1():
    print(num)


def func2():
    num = 666
    print(num)


func1()
func2()
```

也可以在函数内部声明并修改全局变量。

```python
num = 233


def func1():
    num = 666
    print(num)


def func2():
    global num
    num = 666
    print(num)


func1()
print(num)
func2()
print(num)
```
