# 闭包

在嵌套函数中，内部函数使用了外部函数的变量，而外部函数又返回了内部函数，这样的函数称为闭包。

```python
def outer():
    num = 233

    def inner():
        print(num)

    return inner


print(outer())
outer()()
ot = outer()
ot()
```

执行下面的代码，以闭包函数方式传递参数。

```python
def outer(num1):
    print(num1)

    def inner(num2):
        print(num1 + num2)

    return inner


outer(233)(666)
```
