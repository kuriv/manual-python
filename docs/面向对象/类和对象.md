# 类和对象

类属性属于类本身，实例属性属于对象。执行下面的代码，定义类属性与类方法并进行调用。

```python
class Person:
    name = 'Tom'
    age = 20

    def call(self):
        print(self.name, self.age)


print(Person.name)
print(Person.age)
Person.gender = 'male'
print(Person.gender)
p = Person()
p.call()
print(p.name)
print(p.age)
print(p.gender)
```
