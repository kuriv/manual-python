# 字典类型

字典是一种映射类型，它是一种无序键值对的集合。

```python
dict = {'id': 1, 'name': 'Google', 'site': 'www.google.com'}
print(dict)
print(dict['id'])
print(dict['name'])
print(dict['site'])
print(dict.keys())
print(dict.values())
```

也可以通过构造函数的方式来初始化字典。

```python
dict = dict(id=1, name='Google', site='www.google.com')
print(dict)
print(dict['id'])
print(dict['name'])
print(dict['site'])
print(dict.keys())
print(dict.values())
```
