# 单例模式

单例模式是一种创建型设计模式，‌使用单例模式能够确保一个类只有一个实例。

```python
class Singleton(object):
    obj = None

    def __new__(cls, *args, **kwargs):
        if not cls.obj:
            cls.obj = super().__new__(cls)
        return cls.obj

    def __init__(self):
        print('Hello World!')


s1 = Singleton()
print(s1, id(s1))
s2 = Singleton()
print(s2, id(s2))
```
