# 集合类型

集合是一种无序、可变的数据类型，用于存储唯一的元素。

```python
set = {'Google', 'Facebook', 'Google', 'TikTok'}
print(set)
if 'Google' in set:
    print('Found')
else:
    print('Not Found')
```

执行下面的代码，进行并集、交集、差集等常见的集合操作。

```python
set1 = set('adcabde')
set2 = set('cdf')
print(set1 | set2)
print(set1 & set2)
print(set1 - set2)
print(set1 ^ set2)
```

创建一个空集合需要使用以下的方式。

```python
set = set()
print(set)
```
