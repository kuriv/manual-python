# 私有方法

以两个下划线开头声明的方法为私有方法，私有方法只能在类的内部调用，不能在类的外部调用。

```python
class Person:
    name = 'Tom'
    age = 20

    def __init__(self, name, age):
        self.name = name
        self.age = age

    def call(self):
        self.__call()

    def __call(self):
        print(self.name, self.age)


p = Person('Tom', 20)
p.call()
p.__call()
```
