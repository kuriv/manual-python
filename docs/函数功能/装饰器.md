# 装饰器

装饰器是 Python 中的高级功能，主要用来实现动态修改函数或类的行为。

```python
def outer(fn):
    print('outer')

    def inner():
        print('inner before')
        fn()
        print('inner after')

    return inner


@outer
def func():
    print('Hello World!')


func()
```

执行下面的代码，实现多个装饰器堆叠，注意此方式将按照从下到上的顺序依次应用。

```python
def deco1(fn):
    def wrapper():
        print('decorator 1')
        fn()

    return wrapper


def deco2(fn):
    def wrapper():
        print('decorator 2')
        fn()

    return wrapper


@deco1
@deco2
def func():
    print('Hello World!')


func()
```
